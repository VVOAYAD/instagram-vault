# Setup Guide
### No tech experience needed — follow each step in order.

---

## What you're setting up

- You write notes in **Obsidian** → they sync to **GitHub** automatically
- Every day at 9am, GitHub generates your post and publishes it to **Instagram**
- The images are generated by **Replicate** (AI image model, similar to Midjourney)
- The captions are written by **Claude** (your AI content engine)

Total setup time: ~45 minutes.

---

## Step 1 — GitHub (5 min)

GitHub is where your vault lives and where the automation runs. It's free.

1. Go to **github.com** and create a free account
2. Click the **+** button (top right) → **New repository**
3. Name it: `instagram-vault`
4. Set it to **Public** (required — Instagram needs to fetch your carousel images from this repo)
5. Click **Create repository**
6. Click **uploading an existing file**
7. Drag and drop the entire `instagram_system` folder contents
8. Click **Commit changes**

---

## Step 2 — Replicate (3 min)

Replicate generates your AI images. ~$0.003 per image (less than half a cent).

1. Go to **replicate.com** and sign up
2. Click your profile picture (top right) → **API tokens**
3. Click **Create token** → name it "instagram"
4. Copy the token — it starts with `r8_`
5. **Save it somewhere** (you'll need it in Step 6)

---

## Step 3 — Instagram Business Account (5 min)

Your Instagram must be a **Business** or **Creator** account (not personal).

1. Open Instagram → go to your profile
2. Tap the menu (≡) → **Settings and privacy**
3. Tap **Account type and tools** → **Switch to professional account**
4. Choose **Creator** or **Business** → follow the steps
5. When asked to connect a Facebook Page — create one (just use any name, it's required)

---

## Step 4 — Meta Developer App (15 min)

This gives the system permission to post on your behalf.

### 4a. Create the app
1. Go to **developers.facebook.com** and log in with your Facebook account
2. Click **My Apps** → **Create App**
3. Select **Other** → **Next**
4. Select **Consumer** → **Next**
5. Name it: `Instagram Auto Poster` → **Create App**

### 4b. Add Instagram permissions
1. In your app dashboard, find **Instagram Graph API** → click **Set up**
2. In the left menu, click **Tools** → **Graph API Explorer**
3. At the top, select your app from the dropdown
4. Click **Generate Access Token**
5. A popup appears — log in and click **Continue as [your name]**
6. Check these permissions:
   - `instagram_basic`
   - `instagram_content_publish`
   - `pages_read_engagement`
7. Click **Generate Token** → copy it

### 4c. Convert to long-lived token (lasts 60 days)
Open this URL in your browser — replace the 3 values:
```
https://graph.facebook.com/oauth/access_token?grant_type=fb_exchange_token&client_id=YOUR_APP_ID&client_secret=YOUR_APP_SECRET&fb_exchange_token=SHORT_TOKEN
```
- `YOUR_APP_ID` → found in your app's **Settings → Basic**
- `YOUR_APP_SECRET` → same page, click **Show**
- `SHORT_TOKEN` → the token you just copied

Open the URL → you'll see JSON with `"access_token"`. Copy that long token.

### 4d. Get your Instagram User ID
Open this URL in your browser (replace TOKEN with your long-lived token):
```
https://graph.facebook.com/me/accounts?access_token=TOKEN
```
You'll see a `"id"` for your Facebook Page. Then open:
```
https://graph.facebook.com/PAGE_ID?fields=instagram_business_account&access_token=TOKEN
```
The `"id"` inside `instagram_business_account` is your **Instagram User ID** — save it.

---

## Step 5 — Add secrets to GitHub (5 min)

GitHub Secrets keep your API keys safe and pass them to the automation.

1. Go to your `instagram-vault` repo on GitHub
2. Click **Settings** → **Secrets and variables** → **Actions**
3. Click **New repository secret** for each of these:

| Secret Name | Value |
|---|---|
| `ANTHROPIC_API_KEY` | Your Anthropic key (from console.anthropic.com) |
| `REPLICATE_API_TOKEN` | Your Replicate token (starts with `r8_`) |
| `INSTAGRAM_ACCESS_TOKEN` | The long-lived token from Step 4c |
| `INSTAGRAM_USER_ID` | The numeric ID from Step 4d |

---

## Step 6 — Obsidian (5 min)

1. Open Obsidian
2. Click **Open folder as vault**
3. Select your `instagram_system` folder (wherever you saved it)
4. Install the **Obsidian Git** plugin:
   - Settings → Community plugins → Browse → search "Obsidian Git" → Install → Enable
5. In Obsidian Git settings, set **Auto pull interval**: 5 minutes, **Auto push interval**: 5 minutes
6. Set the **remote URL** to your GitHub repo URL (e.g. `https://github.com/yourname/instagram-vault`)

Now when you write a note in Obsidian, it syncs to GitHub automatically.

---

## Step 7 — Test it (3 min)

1. Go to your repo on GitHub
2. Click **Actions** tab
3. Click **Daily Instagram Post** in the left sidebar
4. Click **Run workflow** → **Run workflow**
5. Watch it run — takes about 2-3 minutes total (it generates slides, commits them, waits for GitHub's CDN, then posts)
6. Check your Instagram — a 7-slide carousel should appear

---

## How to write posts from now on

1. Open Obsidian
2. Go to the **Downloads** folder
3. Create a new note — the filename becomes the post title
4. Write your download/reflection as the note body
5. Done — it will post automatically at 9am UTC the next day

**To post immediately:** go to GitHub → Actions → Run workflow manually.

---

## Changing post time

The schedule is set in `.github/workflows/daily_post.yml`.
Find this line and change the time (it's in UTC):
```
- cron: '0 9 * * *'   # 9am UTC
```
`0 9 * * *` = 9:00am UTC. `0 14 * * *` = 2:00pm UTC. `0 7 * * 1` = 7am UTC on Mondays only.

---

## Token refresh (every 60 days)

Instagram tokens expire after 60 days. When yours expires:
1. Repeat Step 4b and 4c
2. Go to GitHub → Settings → Secrets → update `INSTAGRAM_ACCESS_TOKEN`

---

## Troubleshooting

**Post didn't appear** → GitHub → Actions → click the failed run → read the error message

**"Phase 2 failed" / Instagram URL error** → your repo might be set to Private. Go to Settings → Change repository visibility → Make it Public

**Slides look blank or wrong** → add Replicate token to GitHub Secrets (Step 5) — without it, slides won't have AI backgrounds

**Instagram error** → check that your account is Business/Creator and the token has `instagram_content_publish` permission

**"No pending plan found"** → Phase 1 failed before Phase 2 could run. Check the Phase 1 log for the real error.
